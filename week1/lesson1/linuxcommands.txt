ctrl + ALt + T = открытие терминала на убунту

/ - символ корневой директории
~ - символ домашней директории (тильда)

1. pwd (print working directory) = выводит полный путь до текущей папки (директория)

2. ls (list) = выводит список всех папок (директорий) и файлов в текущей рабочей папке (директории)

3. cd (change directory) = меняет текущую рабочую директорию (папку) на указанную (cd <path>)

4. touch = создает указанный файл (touch <имя_файла> [.<имя_файла>] - создать скрытый файл)

5. ls -a = выводит список скрытых и обычных файлов

6. mkdir (make directory) = создает папку (mkdir <name_of_directory>)
*если мы хотим в названии директории был пробел то прописываем '\' -> mkdir Hello\ World

7. rmdir (remove directory) = удаляет директорию (папка) -> rmdir <name_of_dir>

8. rm -rf = принудительное удаление директория (rm -rf <name_of_dir>)

9. rm (remove) = удаляет указанный файл

10. whoami = показывает юзера

11. ls -la = выводит список всех директорий и файлов с подробной инфой 

12. cat <path_to_file> = выводит содержимое файла

13. nano <path_to_file> = открывает графический интерфейс
*если нет указанного файла, то создаться новый файл с таким именем
